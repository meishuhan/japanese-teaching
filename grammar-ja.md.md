# ZIMAの日本語文法

# 日本語の文骨格（Skeleton）

**核心原則**

日本語は条件・修飾語を先に言い、最後に動作・判断を言う。

動詞は常に文末に来る。すべての修飾語は動詞の前に置く。

---

## 一、通用文骨格（最も完全）

```
【時間】
＋【頻度副詞】
＋【対象 / 場所 / 手段】
＋【状態 / 様態】
＋【動詞 / 文末】
```

---

## 二、副詞の順序骨格（デフォルトの安全な順序）

```
頻度 → 程度 → 状態 → 動作
```

---

## 三、副詞の種類と位置

| 副詞の種類 | 役割 | よく来る位置 |
| --- | --- | --- |
| 頻度副詞 | 多い・少ない | 文頭 |
| 程度副詞 | 強い・弱い | 形容詞・動詞の前 |
| 様態副詞 | どのようにするか | 動詞の直前 |
| 否定副詞 | 〜ない | 動詞の直前 |

**位置のルール**

抽象的なものほど前、具体的なものほど動詞に近い。

---

## 四、形容詞と名詞の骨格

### 1. 名詞を修飾する場合

```
形容詞 ＋ 名詞
```

### 2. 述語として使う場合（判断）

```
名詞 ＋ 形容詞（文末）
```

---

## 五、助詞の最小骨格

```
名詞 ＋ 助詞
```

助詞は常に名詞の後ろに付き、順序は変わらない。

---

## 一言方法論まとめ

まず文骨格を確認し、そこに語彙を当てはめる。

文法は暗記するものではなく、活用するもの。

# よく使う「品詞転換ルール」（高頻度）

**一、動詞 → 名詞**

1. 辞書形 + こと（抽象的な事実・経験）
   - 日本語を勉強することは楽しい
2. 連用形 + 方（かた）（方法）
   - 話し方
3. 連用形 + 中（ちゅう）（進行中）
   - 勉強中

**二、動詞 → 副詞・修飾語**

1. て形（方式・順序・同時）
   - 考えて答える
2. 辞書形 + ように（目標・結果）
   - 忘れないようにメモする
3. 普通形 + ほど（程度）
   - 倒れるほど疲れた

**三、形容詞 → 副詞**

1. い形容詞 → い を取って + く
   - 早い → 早く行く
2. な形容詞 → + に
   - 静かに話す

**四、名詞 → 副詞**

1. 名詞 + に（状態・頻度・時間）
   - 毎日に（×）→ 毎日（✓ 固定表現）
   - 急に
   - 本気に
2. 名詞 + で（手段・範囲・状態）
   - 一人で行く
   - 日本語で話す

**五、名詞 → な形容詞**

- 不安 → 不安な気持ち
- 現実 → 現実的な考え

👉 〜的（てき）は超高頻度の「形容詞製造機」

**六、文全体 → 名詞**

1. 普通形 + の（具体的・口語）
   - 走るのが好き
2. 普通形 + こと（正式・抽象）
   - 規則を守ることが重要

# 一言記憶法（重要）

- こと / の：文全体を「一つのもの」にする
- に / で：名詞を「状態・方法」にする
- く / に：形容詞の副詞化
- ほど / ように：「程度・目標」を引き出す

# 口語表現集

## 一、話し出し＆ポーズコントロール（言葉に詰まらない）

> 時間を稼ぎ、リズムを保ち、フリーズを防ぐ

| 表現 | 機能 | 感覚 |
| --- | --- | --- |
| えーと | 考えの間 | うーん… |
| あの | やわらかく始める | あの… |
| まあ | トーンを下げる | まあ… |
| そうですね | 相づち | そうですね |
| ええとですね | 長めの間 | えーとですね… |

核心ルール：

> 間を使いこなす人が、会話の主導権を持つ。

## 二、つなぎ言葉（話を続ける）

> 話を途切れさせない

| 接続詞 | 使い方 | 感覚 |
| --- | --- | --- |
| で | 続ける | それで |
| それで | 結果を受ける | だから |
| そして | 順序で進める | そして |
| それから | 時間順 | その後 |
| また | 補足 | さらに |

口語の黄金公式：

> 文 + で / それで + 次の文

## 三、因果表現（「なぜ」を説明する）

> 人がよく聞くのは「何」ではなく「なぜ」

| 表現 | ニュアンス | 使い方 |
| --- | --- | --- |
| だから | 直接的な因果 | だから〜 |
| そのため | 論理的な因果 | そのため〜 |
| なぜなら | 理由の説明 | なぜなら〜から |
| 〜から | 理由 | 〜から〜 |
| 〜ので | やわらかい理由 | 〜ので〜 |

自然な順序：

> 結論を先に → だから／なぜなら → 理由

## 四、列挙・並列（「読点感覚」）

> 実際の会話 ＝ かたまりをどんどん出す

| 表現 | 使い方 | 感覚 |
| --- | --- | --- |
| AやB | 並列 | AとB |
| AとかB | 例示 | たとえばAとかB |
| Aなど | 締め | 〜など |
| AもBも | 全部 | どちらも |
| AたりBたり | 行為の列挙 | AしたりBしたり |
| AしBし | 複数の理由 | AだしBだし |

話し言葉の本質：

> 完全な文ではなく、並列の積み重ね。

## 五、言い直し・訂正（人間なら必ずする）

> 「言い間違えたら終わり」をなくす

| 表現 | 機能 | 感覚 |
| --- | --- | --- |
| というか | 言い直す | というか〜 |
| 正確には | 精確にする | 正確には〜 |
| どちらかというと | ぼかす | どちらかというと〜 |
| 少し違うけど | 訂正する | ちょっと違うんですが |

## 六、仮定・探り・余地を残す

> 社交の安全地帯

| 表現 | 使い方 | 感覚 |
| --- | --- | --- |
| もし〜たら | 仮定 | もし〜たら |
| 〜なら | 前提 | 〜なら |
| 〜かもしれない | 不確定 | 〜かもしれない |
| 〜と思う | 主観 | 〜と思う |
| 〜と思います | 丁寧 | 〜と思います |

成熟した表現の核心：

> 常に逃げ道を残しておく。

## 七、まとめ・締め（話を閉じる）

> 相手に「わかった」と感じさせる

| 表現 | 機能 | 感覚 |
| --- | --- | --- |
| つまり | まとめ | つまり〜 |
| 要するに | 圧縮 | 要するに〜 |
| 結果として | 結果 | 結果として〜 |
| まとめると | 締め | まとめると〜 |

## 八、実際の話し方「連用型フロー」

テンプレート1｜思考型

えーと → つまり → 結論

テンプレート2｜説明型

あの → なぜなら → 〜から → だから

テンプレート3｜列挙型

AとかBとか → など → まとめると

テンプレート4｜安全社交型

まあ → 〜と思います → もし〜なら

**この方法全体の核心一言：**

> 日本語は文法ではなく：ポーズ → つなぐ → 列挙 → 因果 → まとめる

これは「話し方の流れ」であり、知識の羅列ではない。

# 文法分類表

## A. 状態・習慣・傾向

### A0 基本状態・動作のつながり（N5核心）

| 文法 | 意味 | 例文 | 公式・備考 |
| --- | --- | --- | --- |
| 〜て | 〜して、それから… | ご飯を食べ**て**寝る。 | 動詞て形 |
| 〜てから | 〜した後で… | 家に帰っ**てから**勉強する。 | 時間順序 |
| 〜ながら | 〜しながら… | 音楽を聞き**ながら**作業する。 | 同時進行 |
| 〜まえに | 〜する前に | 寝る**前に**シャワーを浴びる。 | 時間の前後 |
| 〜のが好きだ | 〜するのが好き | 音楽を聞く**のが好きだ**。 | V辞書形＋のが好き |

### A1 進行中・状態の継続

| 文法 | 意味 | 例文 | 公式・備考 |
| --- | --- | --- | --- |

| 〜てしまう | うっかり〜してしまう / 〜し終える | 宿題を忘れ**てしまった**。 | Vて＋しまう |

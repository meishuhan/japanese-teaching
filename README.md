## 文档结构

### 🦴 第一部分：句子骨架
日语的核心逻辑：**动词永远在句尾，所有修饰都在前。**

先建立这个骨架，后面所有语法都是往里填词。

### 🔄 第二部分：词性转换规则
动词 → 名词 → 副词 → 形容词……

掌握转换规则，一个词可以在句子里变出多种用法。

### 💬 第三部分：口语废话大全
真实说话不是完美语法，是：
- 开口不崩（停顿控制）
- 把话说下去（连接词）
- 改口 / 留余地 / 总结收束

### 📋 第四部分：语法分类大表（A～G）

按**功能**分类，不按 JLPT 等级分类：

| 板块 | 内容 |
|------|------|
| A | 频率・习惯・状态 |
| B | 意志・计划・决定 |
| C | 原因・理由・结果 |
| D | 对比・让步・限制 |
| E | 推量・判断・可能性 |
| F | 时间顺序・阶段・变化 |
| G | 对象・范围・依据・手段 |

每条语法都包含：**中文含义 / 日文例句 / 中文解释 / 套用公式**

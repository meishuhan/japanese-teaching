## 中文

> 不是背语法，是套用语法。先有骨架，再填内容。

### 这份文档适合谁？

- 有一定基础单词量（日常词汇）
- 想跳过低效的死记硬背，直接掌握语法框架
- 目标快速提升到 **N2 使用水平**
- 理解能力强，能接受高密度输入

> ⚠️ 具体使用方法将在后续更新中补充。

### 文档结构

| 部分 | 内容 |
|------|------|
| 第一部分 | 句子骨架——动词永远在句尾，所有修饰都在前 |
| 第二部分 | 词性转换规则——一个词变出多种用法 |
| 第三部分 | 口语废话大全——开口不崩、因果、连接、收束 |
| 第四部分 | 语法分类大表 A～G——N5 到 N2 按功能分类 |

### 如何参与

- ✅ 可以修改：例句错误、语法说明不准确、中文解释不清晰
- ❌ 请勿改动：整体框架、A～G 分类逻辑、极简排版风格

**参与方式：** Fork → 修改 → Pull Request，说明改了什么、为什么。

## 日本語

> 文法は暗記するものではなく、活用するもの。まず骨格を作り、そこに語彙を当てはめる。

### 対象読者

- 基礎的な語彙がある（日常単語を知っている）
- 丸暗記を避け、文法の骨格を理解したい
- 目標は **N2レベル** の運用力
- 理解力が高く、高密度インプットに対応できる

> ⚠️ 具体的な使用方法は今後のアップデートで補足予定。

### ドキュメント構成

| パート | 内容 |
|--------|------|
| 第一部 | 文の骨格——動詞は常に文末、修飾語はすべて前に |
| 第二部 | 品詞転換ルール——一つの語を多様に使いこなす |
| 第三部 | 口語表現集——話し出し・因果・つなぎ・まとめ |
| 第四部 | 文法分類表 A～G——N5からN2まで機能別に整理 |

### 参加方法

- ✅ 修正可能：例文の誤り、文法説明の不正確さ、説明が不明瞭な箇所
- ❌ 変更不可：全体の骨格構造、A～Gの分類ロジック、シンプルなレイアウトスタイル

**参加手順：** Fork → 修正 → Pull Request（何を・なぜ変えたかを記載）

## English

> Don't memorize grammar — apply it. Build the skeleton first, then fill in the words.

### Who is this for?

- Has basic vocabulary (everyday words)
- Wants to skip rote memorization and grasp grammar frameworks directly
- Target: **N2 level** practical ability
- Strong learner who can handle high-density input

> ⚠️ Detailed usage guide coming in future updates.

### Document Structure

| Part | Content |
|------|---------|
| Part 1 | Sentence Skeleton — verb always last, all modifiers come before |
| Part 2 | Word Class Conversions — make one word work in many ways |
| Part 3 | Spoken Japanese Toolkit — openers, cause & effect, connectors, wrap-ups |
| Part 4 | Grammar Index A～G — N5 to N2 organized by function |

### How to Contribute

- ✅ Feel free to fix: example sentence errors, inaccurate grammar explanations, unclear wording
- ❌ Please don't change: overall framework structure, A～G classification logic, minimalist layout style

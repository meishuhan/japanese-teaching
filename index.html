第一部分：句子骨架

日语核心原则：动词永远在句尾，所有修饰都在动词前。

通用句子骨架（最完整）

【时间】 ＋【频率副词】 ＋【对象 / 场所 / 手段】 ＋【状态 / 样态】 ＋【动词 / 句尾】

副词排序骨架

频率 → 程度 → 状态 → 动作

口诀：越抽象越靠前，越具体越靠近动词。

副词类型与位置

副词类型作用常见位置
频率副词多不多句首
程度副词强不强形容词/动词前
状态副词怎么做动词正前
否定副词不动词正前

整体学习顺序

顺序内容
1句子骨架
2词性转换
3常见副词
4具体语法点

第一部：文の骨格

日本語の核心：動詞は常に文末、すべての修飾語は動詞の前。

基本文の骨格（完全版）

【時間】 ＋【頻度副詞】 ＋【対象 / 場所 / 手段】 ＋【状態 / 様態】 ＋【動詞 / 文末】

副詞の順序

頻度 → 程度 → 状態 → 動作

ルール：抽象的なものほど前に、具体的なものほど動詞の近くに。

副詞の種類と位置

副詞の種類役割主な位置
頻度副詞頻度を表す文頭
程度副詞程度を表す形容詞・動詞の前
様態副詞様子を表す動詞の直前
否定副詞否定動詞の直前

第二部：品詞転換ルール

動詞 → 名詞

公式意味例

形容詞 → 副詞

種類ルール例

Part 1: Sentence Skeleton

Core principle: The verb always comes last. Everything else modifies the verb and comes before it.

Full sentence skeleton

【Time】 ＋【Frequency adverb】 ＋【Object / Place / Means】 ＋【State / Manner】 ＋【Verb / Sentence ending】

Adverb ordering

Frequency → Degree → Manner → Action

Rule: The more abstract, the earlier it comes. The more specific, the closer to the verb.

Adverb types and positions

TypeFunctionPosition
FrequencyHow oftenSentence start
DegreeHow muchBefore adj/verb
MannerHow it's doneRight before verb
NegationNotRight before verb

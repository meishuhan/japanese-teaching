# ZIMA's Japanese Grammar

# Japanese Sentence Skeleton

**Core principle**

Japanese puts conditions and modifiers first, action / judgment last.

The verb always comes at the end. All modifiers come before the verb.

---

## 1. Full sentence skeleton

```
[Time]
+ [Frequency adverb]
+ [Object / Place / Means]
+ [State / Manner]
+ [Verb / Sentence ending]
```

---

## 2. Adverb ordering skeleton (safe default order)

```
Frequency → Degree → Manner → Action
```

---

## 3. Adverb types and common positions

| Adverb type | Function | Common position |
| --- | --- | --- |
| Frequency adverb | How often | Start of sentence |
| Degree adverb | How much / how strong | Before adjective / verb |
| Manner adverb | How it's done | Right before verb |
| Negation adverb | Not | Right before verb |

**Position rule**

The more abstract, the earlier. The more specific, the closer to the verb.

---

## 4. Adjective and noun skeletons

### 1. Modifying a noun

```
Adjective + Noun
```

### 2. As a predicate (judgment)

```
Noun + Adjective (at sentence end)
```

---

## 5. Minimum particle skeleton

```
Noun + Particle
```

Particles always follow the noun — the order never changes.

---

## 6. Overall learning order (structure first)

| Order | Content |
| --- | --- |
| 1 | Sentence skeleton |
| 2 | Word class conversions |
| 3 | Common adverbs |
| 4 | Specific grammar points |

---

## One-line methodology

Establish the sentence skeleton first, then fill in the words.

Grammar is not memorized — it is applied.

# Common Word Class Conversion Rules (High Frequency)

**I. Verb → Noun**

1. Dictionary form + こと (abstract fact / experience)
   - 日本語を勉強することは楽しい → Studying Japanese is fun
2. Stem + 方（かた） (method)
   - 話し方 → Way of speaking
3. Stem + 中（ちゅう） (in progress)
   - 勉強中 → Currently studying

**II. Verb → Adverb / Modifier**

1. て-form (manner / sequence / simultaneous)
   - 考えて答える → Think and then answer
2. Dictionary form + ように (goal / result)
   - 忘れないようにメモする → Take notes so you don't forget
3. Plain form + ほど (degree)
   - 倒れるほど疲れた → So tired I could collapse

**III. Adjective → Adverb**

1. い-adjective → drop い + く
   - 早い → 早く行く → Go quickly
2. な-adjective → + に
   - 静かに話す → Speak quietly

**IV. Noun → Adverb**

1. Noun + に (state / frequency / time)
   - 毎日に（×）→ 毎日（✓ fixed form）
   - 急に → Suddenly
   - 本気に → Seriously
2. Noun + で (means / scope / state)
   - 一人で行く → Go alone
   - 日本語で話す → Speak in Japanese

**V. Noun → な-adjective**

- 不安 → 不安な気持ち → Anxious feeling
- 現実 → 現実的な考え → Realistic thinking

👉 〜的（てき） is an ultra-high-frequency "adjective factory"

**VI. Whole sentence → Noun**

1. Plain form + の (concrete / casual)
   - 走るのが好き → I like running
2. Plain form + こと (formal / abstract)
   - 規則を守ることが重要 → It is important to follow the rules

# One-line memory tips

- こと / の: turns a whole sentence into "a thing"
- に / で: turns a noun into "a state or method"
- く / に: adjective → adverb
- ほど / ように: pulls out "degree / goal"

# Spoken Japanese Toolkit

## 1. Openers & Pause Control (Don't freeze up)

> Buy time, hold the rhythm, avoid going blank

| Expression | Function | English feel |
| --- | --- | --- |
| えーと | Thinking pause | Umm… |
| あの | Soft opener | Well… |
| まあ | Tone softener | I mean… |
| そうですね | Backchanneling | I see… right… |
| ええとですね | Extended pause | Ummm… so… |

Core rule:

> The person who masters pausing controls the conversation.

## 2. Continuing (Keep the talk going)

> Don't let the conversation drop

| Connector | Usage | English |
| --- | --- | --- |
| で | Continue | And then |
| それで | Build on result | So |
| そして | Sequence | And then |
| それから | Time order | After that |
| また | Add more | Also |

Spoken golden formula:

> Sentence + で / それで + next sentence

## 3. Cause & Effect (Explain "why")

> People ask "why" more than "what"

| Expression | Feel | English |
| --- | --- | --- |
| だから | Direct cause | So / Therefore |
| そのため | Rational | For that reason |
| なぜなら | Explaining cause | Because |
| 〜から | Reason | Because… |
| 〜ので | Soft reason | Since… (gentle) |

Natural order:

> Conclusion first → だから／なぜなら → Reason

## 4. Listing (The "comma feeling")

> Real speech = dumping things out in chunks

| Expression | Usage | Feel |
| --- | --- | --- |
| AやB | Listing | A and B |
| AとかB | Examples | Like A or B |
| Aなど | Closing | …and so on |
| AもBも | All of them | Both A and B |
| AたりBたり | Listing actions | Sometimes A, sometimes B |
| AしBし | Multiple reasons | Both A and B (reasons) |

Core insight:

> Real speech isn't complete sentences — it's parallel stacking.

## 5. Correcting Yourself (Humans do this constantly)

> Stops you from dying when you say the wrong thing

| Expression | Function | English |
| --- | --- | --- |
| というか | Walk back | Or rather |
| 正確には | Sharpen | More precisely |
| どちらかというと | Soften | I'd say more… |
| 少し違うけど | Redirect | Not quite but… |

## 6. Hedging / Leaving Room

> Your social safety zone

| Expression | Usage | English |
| --- | --- | --- |
| もし〜たら | Hypothesis | If… |
| 〜なら | Premise | Given that… |
| 〜かもしれない | Uncertainty | Maybe |
| 〜と思う | Subjective | I think |
| 〜と思います | Polite | I believe |

Mature speech core:

> Always leave yourself a back door.

## 7. Wrapping Up (Close the loop)

> Make the other person feel "I get it"

| Expression | Function | English |
| --- | --- | --- |
| つまり | Summarize | In other words |
| 要するに | Compress | Simply put |
| 結果として | Result | In the end |
| まとめると | Close | To sum up |

## 8. Real Speech Flow Templates

> The "human speaking model" you actually need

Template 1 | Thinking type

えーと → つまり → Conclusion

Template 2 | Explanation type

あの → なぜなら → 〜から → だから

Template 3 | Listing type

AとかBとか → など → まとめると

Template 4 | Safe social type

まあ → 〜と思います → もし〜なら

**The one-line core of this whole method:**

> Japanese isn't grammar — it's: Pause → Connect → List → Cause → Conclude

This is a speaking flow, not a list of knowledge points.
